# GarageBook Outreach 2026

> Living document. Version 0.1

## Doel

Dit document beschrijft de volledige outreachstrategie voor GarageBook. Het is nadrukkelijk een werkdocument dat steeds verder wordt uitgebreid.

## Uitgangspunten

- Hergebruik de bestaande outreachmodule.
- Geen handmatige één-op-één acquisitie waar automatisering mogelijk is.
- Segment-specifieke campagnes.
- Eén bronbestand voor prospects.
- Alles meetbaar.

## Doelgroepen (eerste indeling)

1. Motorclubs
2. Koepelorganisaties
3. Evenementen
4. Trackdays
5. Toerorganisaties
6. Motorvakanties
7. Opleidingen
8. Webshops
9. Verzekeraars
10. Media
11. Musea
12. Klassiekerclubs
13. Merkspecifieke clubs
